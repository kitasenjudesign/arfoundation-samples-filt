VirtualDevice



Description

An abstraction for a virtual (software-based) audio input device.



Public Functions

Dispose Dispose the virtual device and release resources.
StartRecording Start recording from the audio device.
StopRecording Stop recording from the audio device.


Constructors

VirtualDevice Create a virtual audio device backed by an AudioSource.
VirtualDevice Create a virtual audio device backed by an AudioListener.


Public Variables

IsRecording Is the device currently recording?
