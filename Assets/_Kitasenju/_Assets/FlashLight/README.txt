Thank for purshasing iOS Native Flashlight !

+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+

You will find in this package 2 folders.

In the Plugins folder there is native code written in Objective C for iOS.
(Do not edit these files if you don't know how to code in Objective C)

In the Scripts folder you will find the FlashlightController.cs
This is the main script of this asset. You can control your flashlight with this. 
Just attach it to a GameObject or spawn the prefab "FlashlightController" and it's done.

You can look in the demo scene how to use it.
